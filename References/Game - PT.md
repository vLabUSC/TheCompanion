**Developer:** Kojima Productions | **Year:** 2014 | **Platform:** PS4 (delisted)

P.T. (Playable Teaser) is a first-person horror demo created as a teaser for a cancelled *Silent Hill* collaboration between Hideo Kojima and Guillermo del Toro. The player walks through an L-shaped hallway that loops endlessly, each pass revealing new details.
