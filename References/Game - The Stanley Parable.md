## What It Is
A first-person exploration game in which a narrator describes everything Stanley — you — does, in past tense, as if it's already written. You can follow the narration or defy it; each branch is a different short ending, and the game keeps acknowledging that it knows you're playing it. A piece about authorial control, illusion of choice, and what a game is.
