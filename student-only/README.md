# Your Personal Working Folder

This folder is yours. The Companion writes here when you ask it to save something — a project plan, a note, a reference you want to remember, a question for your professor.

What lives here:

- **`where-we-left-off.md`** — A short paragraph the Companion updates at the end of each session. When you start the next session, it'll read this and offer to pick up where you left off.
- **`notes/`** — Free-form notes by topic. Ask the Companion to "remember this as a note" and it'll create or append a file here.
- **`references/`** — Games or films you want to save as your own examples. Same idea as the bundle's `References/` folder, but yours.
- **`projects/`** — Saved project plans (one file per project). The Companion offers to save these at the end of a project-planning session.
- **`instructor-questions.md`** — A running list of things you want to bring up with your professor. The Companion appends here when you say "remember this to ask my professor."

You can edit any of these files yourself. Open them in any text editor or in the Companion.
