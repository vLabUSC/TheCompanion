#### Node Pin Types

- Execution Pins (White): Control flow direction - when code executes in sequence
- Boolean Pins (Red): True/false values for conditional logic and state checks
- Integer Pins (Cyan): Whole numbers for counters, array indices, and discrete quantities
- Float Pins (Green): Decimal numbers for health, speed, distances, and smooth interpolation
- String Pins (Pink): Text data for names, messages, and file paths
- Vector Pins (Gold): 3D coordinates (X, Y, Z) for positions, directions, and velocities
- Rotator Pins (Purple): 3D rotation values (Pitch, Yaw, Roll) for object orientation
- Object Reference Pins (Blue): References to actors, components, and other Blueprint objects
- Wildcard Pins (Gray): Accept any data type - used in generic functions
